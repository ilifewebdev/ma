# My Pony Adventure (小马宝莉大冒险)

A cute side-scrolling runner game featuring "My Little Pony" characters!

## Features

- **Endless Runner**: Run, jump, and slide to avoid obstacles.
- **Characters**: Play as Applejack, Rainbow Dash, Twilight Sparkle, and more!
- **Level System**: 10 unique levels with different themes (Meadow, Beach, Space, etc.).
- **Account System**: Save your progress with a nickname.
- **Daily Calendar**: Track your daily runs and achievements.
- **Sharing**: Generate cool result cards to share with friends.

## How to Play

- **Jump**: Tap screen / Spacebar / Up Arrow
- **Double Jump**: Tap again in mid-air
- **Slide**: Swipe down / Down Arrow
- **Move**: Left/Right Arrows

## Deployment

### GitHub Pages

1. Create a new repository on GitHub.
2. Run the deployment script:
   - Windows: `deploy_github.bat`
   - Mac/Linux/Git Bash: `./deploy_github.sh`
3. Go to Repository Settings -> Pages.
4. Select `main` branch and `/ (root)` folder.
5. Save.

Your game will be live at `https://<username>.github.io/<repo-name>/`
